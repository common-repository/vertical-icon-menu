Easily create a fixed vertical icon menu that adjusts its size for smaller screens.

== Description ==
The Wordpress plugin "Vertical Icon Menu"  is designed to create the vertical menu with icon elements. You can implement the compact navigation bar with it. It will be the new and informative solution that well complementing the design of the site. 

= Demo =
[Demo](https://wow-estore.com/item-preview/vertical-icon-menu/)

[youtube https://www.youtube.com/watch?v=oPZR7p85z1M]

= Main features =
* Select menu and icon size
* Select fontawesome icon for button: More then 600+ icons
* Create 1 vertical icon menu
* Insert any link


= Vertical Icon Menu can be used for: = 
* Site navigation
* Contacts block
* User profile navigation
* Skype menu
* And more...

== Installation ==
* Installation option 1: Find and install this plugin in the `Plugins` -> `Add new` section of your `wp-admin`
* Installation option 2: Download the zip file, then upload the plugin via the wp-admin in the `Plugins` -> `Add new` section. Or unzip the archive and upload the folder to the plugins directory `/wp-content/plugins/` via ftp
* Press `Activate` when you have installed the plugin via dashboard or press `Activate` in the in the `Plugins` list 
* Go to `Vertical Icon Menu` section that will appear in your main menu on the left
* Click `Add new` to create your first menu
* Setup your menu
* Click Save
